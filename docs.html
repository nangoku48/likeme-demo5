🧩 プロジェクト概要

コンセプト

SNPITのSnap-to-Earnモデルを踏襲し、「写真 → セルフィー動画」に転換。
ユーザーの“本人性”を重視し、セルフィー投稿 × バトル × SNS連携 × Earn のシンプルな構成。

プロジェクト目的

  • SNS投稿文化を「報酬化」する
  • 若年層向けに「動画 × Earn × 出会い要素」を融合
  • 低予算から開始し、NFT初期販売で資金を確保

🔩 コア機能一覧

機能カテゴリ内容
ユーザー登録Wallet接続 / SNS連携（X, Insta, TikTok）
撮影機能アプリ内カメラで10秒動画撮影（AI検知で本人判定）
投稿撮影動画をバトルへ送信
バトル2動画を自動マッチング→投票数で勝敗決定
トークンLMP（ポイント）/ LMT（ERC-20トークン）
NFTCamera NFT（ERC-721）
セーフガードAI生成検出 / 年齢制限 / 不正投稿BAN

🚀 ロードマップ

  • Phase 1 (0-3ヶ月): NFT販売・テスト版App開発
  • Phase 2 (4-6ヶ月): β版リリース（撮影・投稿・バトル）
  • Phase 3 (6-9ヶ月): トークン導入・ポイント変換実装
  • Phase 4 (9-12ヶ月): SNSキャンペーン・スポンサー導入
  • Phase 5 (12ヶ月〜): DAO構想・トークン上場
